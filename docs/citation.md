# Citation

If you use this fork, please cite **both** this package and the original
TopOMetry work. Machine-readable metadata lives in the repository's
[`CITATION.cff`](https://github.com/HauserGroup/topometryNoSC/blob/master/CITATION.cff)
file — use the "Cite this repository" button on GitHub, or the citation for the
tagged release you used.

## Upstream credit

This package began as a fork of the original
[TopOMetry](https://github.com/davisidarta/topometry) project by David S Oliveira,
Ana I Domingos and Licio A Velloso. Original copyright and the MIT license are
preserved. The fork is independently maintained and is **not** an official
upstream release.

## Original publication

> Oliveira DS, Domingos AI, Velloso LA. *TopoMetry systematically learns and
